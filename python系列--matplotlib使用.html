matplotlib介绍¶

matplotlib是一个出色的基于python的2d图形绘制库，主要有以下优点：

对于简单的绘制特别容易上手
对于复杂的绘制可定义图形中的任意元素
支持多格式高质量的图形输出
可封装拓展，seaborn就是建立在matplotlib的基础上

通常使用 import matplotlib.pyplot as plt导入库文件来使用 在jupyter notebook中使用时添加 %matplotlib inline可使图形在notebook界面中显示

基本绘制方式¶

In [1]:
import numpy as np
import matplotlib.pyplot as plt
x=np.linspace(0,5,11)

#使用plot方法绘制单个曲线图
plt.plot(x,x**2)
#设置x轴标题
plt.xlabel('x value')
#设置y轴标题
plt.ylabel('x square value')
#添加图形标题
plt.title('the plot of x square')
#显示图形
plt.show()

plt.plot(x**2, x, 'y--')

面向对象的绘制方式¶

在绘制多个子图时通常使用基于对象的方式，通过创建一个figure对象，之后调用对象的方法或属性来实现多个图形的绘制

In [6]:
#在同一Figuer绘制多个Axes

#创建figure对象
fig = plt.figure()
#添加两个Axes实例
axes1 = fig.add_axes([0.1, 0.1, 0.8, 0.8]) #列表值表示Axes在Figure中的位置（左下宽高）
axes2 = fig.add_axes([0.2, 0.5, 0.4, 0.3]) 

# 在Axes1绘制图形
axes1.plot(x, x**2, 'r')
axes1.set_xlabel('x value')
axes1.set_ylabel('x square value')
axes1.set_title('plot of x square')

# 在Axes2绘制图形
axes2.plot(x, np.sqrt(x), 'y')
axes2.set_xlabel('x square value')
axes2.set_ylabel('x value')
axes2.set_title('plot of x sqrt')

plt.show()
In [7]:
#在同一Axes绘制多个曲线
fig = plt.figure()
ax = fig.add_axes([0,0,1,1])
ax.plot(x, x**2, label="x**2")
ax.plot(x, x**3, label="x**3")
#显示各曲线图例
ax.legend()
plt.show()

#创建多个Axes的快速方式

#创建1行3列的6个Axes
fig,axes=plt.subplots(nrows=1,ncols=3,figsize=(12, 4))

axes[0].plot(x,x**1,'r')
axes[1].plot(x,x**2,'b')
axes[2].plot(x,x**4,'y')
plt.show()
