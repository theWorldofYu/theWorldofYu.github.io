数据集包含30多万条房屋数据，每条数据有26个特征，各特征的含义如下：

URL : 有关房屋信息的网络地址
ID : 在平台上对应的房屋编号
Lng : 房屋位置信息-经度
Lat : 房屋位置信息-纬度
CommunityID : 房屋所属小区的编号
Followers : 房屋的关注人数
Total price : 房屋总价格
Price : 房屋每平方价格
Square : 房屋面积
LivingRoom : 卧室数量
DrawingRoom : 客厅数量
Kitchen : 厨房数量
Bathroom : 卫生间数量
Floor : 房屋所处楼层
BuildingType : 住宅的建筑类型(塔楼，板楼，塔板结合)
ConstructionTime : 建筑时间
RenovationCondition : 装修情况(简装,精装,其他等)
BuildingStructure : 建筑类型(钢混,混合,砖混,砖木等)
LadderRatio : 所处楼层梯户比
Elevator : 是否有电梯
PropertyRights : 房屋年限是否满五年
Subway : 是否紧邻地铁
District : 所在区域(朝阳,昌平,东城,顺义,西城等)
Community average : 所在小区房屋均价
YearOfTradeTime :成交的年份
MonthOfTradeTime :成交的月份

对成交量进行分析¶

链家算是二手房交易领域的龙头了，基本上每个小区附近都会有家店面，我很好奇如此高覆盖率的店面其成交量如何、影响成交量的因素有哪些

链家在2010年正式上线，我们首先看下至成立以来每年的交易量如何

不同年份成交量¶

In [22]:
#提取年份和各年份成交数量
years=sorted(data['YearOfTradeTime'].unique())

截至2016年，成交量都趋于增长，尤其是15年成交量6.9万套，是14年的2倍，到了16年更突破9万套
到了2017年全年成交量只有4.3万套，相较去年9万套环比下降50%　　

造成如过上车般的大起大落，我想知道是链家内部原因还是外界环境所影响？经过网上调查发现近两年的房产政策有过调整，比如政府2015.3.30日出台房产新政来刺激楼市，到了2017.3.17又出新政来打压楼市。这或许是造成15年成交量大增或17年大减的原因，为了验证这一点，我们可以看下这两年每月的成交量，看两者的时间线是否吻合

15年与17年各月成交量¶

In [23]:
#定义月份并提取15年与17年各月成交量
months=np.arange(1,13)
sort_15=(data[data['YearOfTradeTime']==2015]['MonthOfTradeTime']).value_counts()
sort_17=(data[data['YearOfTradeTime']==2017]['MonthOfTradeTime']).value_counts()

#绘制关系图
plt.figure(figsize=(15,7))
plt.plot(months,[sort_15[month] for month in months],c='blue',label='2015')
plt.plot(months,[sort_17[month] for month in months],c='red',label='2017')
plt.xlabel('month')
plt.ylabel('number of trades ')
plt.legend()
plt.show()

2015年，3.30政策出台的当月，月度交易量突破了6000，且之后5个月一直维持该水平
2017年同样在317新政出台后，由于对二手房交易的调控，成交量由原来的8000多断崖式的跌倒2000，且今后几个月增长平缓
看来我们的猜想是对的，可见房产交易很大程度上受政策的影响

对房价进行分析¶

令我好奇的是为什么都在2环或4环内，西城的均价比东城高，同样海淀的均价也比朝阳高，网络搜索后发现北京最好的小学和中学都分布在西城和海淀，教育资源的优势提升了房屋的价值

大兴、通州、昌平、门头沟均价都在4.5万左右的，令我感兴趣的是通州确定为北京副中心后，房价的变化如何

对房屋总价影响最大的毫无疑问是房屋面积，有0.7的强相关性
意外的是相较于卧室数量，卫生间的数量对房屋总价的影响更大些，会多出3个百分点
另外小区均价，单平米价格及客厅数量对房屋总价有点中到强度的影响
电梯数量，周边是否紧邻地铁及厨房数量也有着10%～20%的影响
