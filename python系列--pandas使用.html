pandas库介绍¶

pandas库是python编程中一个高性能，易于使用的数据结构和数据分析工具

主要用于数据读取、缺失值填补、数据分组、拼接、分层数据索引等操作

通常以import pandas as pd来导入使用，只要数据类型包括Series和DataFrame

Series使用¶

In [1]:
#使用之前导入相关的库文件
import numpy as np
import pandas as pd
In [5]:
#创建一个Series，可以通过list,np.array，dictionary来创建

#使用list
my_list=[1,2,3,4]
labels=['a','b','c','d']
pd.Series(data=my_list,index=labels)
Out[5]:
a    1
b    2
c    3
d    4
dtype: int64
In [7]:
#使用np.array来创建
arr=np.array([10,20,30,40])
pd.Series(data=arr,index=labels)
Out[7]:
a    10
b    20
c    30
d    40
dtype: int32
In [9]:
#使用dictionary来创建
my_dict={'a':10,'b':20,'c':30,'d':40}
pd.Series(my_dict)
Out[9]:

#通过index来获取相应的值
