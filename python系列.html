列表解析式¶

列表解析式提供了一个快速生成列表的方法，不失可读性的同时减少了代码量

In [44]:
#原始代码
a=[]
for i in range(11):
    if i%2==0:
        a.append(i)
print(a)    

[0, 2, 4, 6, 8, 10]
In [47]:
#使用列表解析式的代码
print([x for x in range(11) if x%2==0])

[0, 2, 4, 6, 8, 10]

lambda表达式， map与fiter函数¶

lambda表达式属于匿名函数，省去了声明和命名的过程而直接使用

map函数有两个参数：映射函数和序列，会返回参数函数对序列的映射

filter函数也有两个参数：判断函数和序列，会返回满足判断函数的元素列表

一般情况下lambda表达式会结合map函数或filter函数使用

In [49]:
#获取list1各元素2倍的列表
list1=[1,2,3,4,5]
list(map(lambda x: x*2,list1))
Out[49]:
[2, 4, 6, 8, 10]
In [51]:
#获取list1中是偶数的列表
list(filter(lambda x: x%2==0,list1))
Out[51]:
[2, 4]
